La idea es simular algunas funcionalidades de un sistema de gestión de una pequeña biblioteca. Esta tiene pocos socios, registrados por nro. de DNI, y libros distribuidos en distintos lugares físicos (acá denominados locaciones). Como es un proyecto de prueba, se usa el código de ISBN como identificador de un libro (en lugar de generar un número de inventario o utilizar estándares internacionales).
